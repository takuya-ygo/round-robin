# リーグ戦管理アプリ

ブラウザ上で動作する簡易リーグ戦管理アプリです。

## 機能

- 参加人数可変
- 参加者名変更機能
- 対戦組み合わせの生成機能（ラウンドロビン方式）
- 対戦結果の入力・修正機能
- 総合結果の順位表の表示
- 順位を決定する際の優先する要素の追加削除・順番変更機能
- 対戦結果一覧の表示（マトリックス形式）
- 大会タイトルの設定
- 結果を画像として保存・共有

## 使い方

1. [GitHub Pages](https://takuya-ygo.github.io/round-robin/) にアクセス
2. 参加者を追加
3. 対戦組み合わせを生成
4. 各対戦の結果を入力
5. 順位表で結果を確認

## 技術スタック

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- html2canvas (画像生成用)
